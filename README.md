## Apollo Digitalis Virtualis Dashboard

**NEW**: Interactive VR/AR Dashboard with Euystacio Holographic Chat Interface (CIC)

### Quick Start

```bash
# Option 1: Run demo (starts backend + frontend)
python3 demo_apollo.py

# Option 2: Manual start
# Terminal 1: Backend
python3 oi_server.py

# Terminal 2: Frontend
python3 -m http.server 8080
# Open http://localhost:8080/apollo-dashboard.html
```
